1.入坑即踩坑

1.1 git默认开了代理导致下载失败

git config –global –unset http.proxy 默认开了全局代理，卡了一个月。。。

1.2 url不能含有空格

在windows电脑中，文件名常常含有空格，但在网络url中不能含有空格，负责会解码成%20造成路径错误

cd blog 进入博客目录

hexo clean 清楚生成的public静态文件

hexo g 生成public静态文件

hexo s 启动hexo服务器
